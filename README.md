# Face Recognition Based Student Verification System

## 📌 Overview
This project is an AI/ML based system that verifies student identity using face recognition technology.

## 🚀 Features
- Face Detection using OpenCV
- LBPH Face Recognition
- Real-time Verification
- Image Preprocessing
- High Accuracy (~98%)

## 🛠 Technologies Used
- Python
- OpenCV
- LBPH Algorithm
- Tkinter (GUI)

## 📊 Performance
- Accuracy: 98%
- Real-time detection (<1 sec)

- Project Report

- Source Code
